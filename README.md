## Code You January 2026
## Synopsis
This repo stands to be record of in-class activities and coding demonstrations. Please reach out to your mentors if you have any questions.

## Weekly Classes
A class of the Code You Web Dev course meets weekly on Thursday. Ideally, if a mentor or student goes over a problem or would like to see an example of a topic, this repo can keep track of such activities. 

## Contribution
If something is missed or disfunctional, students are encourged utilize the [Pull Request process](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request) to correct the record or add pieces that were missed by the mentors. Students are encourges to pull down changes to this repo frequently and experiment with class demos.
